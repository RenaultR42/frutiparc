MotionBall2 Generator Help :

  • Creer les levels avec mb2edit
  • Les mettre dans un fichier pour assemblage (voir format ci dessous)
  • Lancer mb2gen.exe fichier_dungeon.txt pour générer le dungeon
Format du fichier d'assemblage :

  • Une ligne par salle
  • 64 lignes en tout (8x8)
  • Données sur les lignes :
    • DATA=mb2edit data : données d'une salle
    • START=... : indique la salle d'entrée du dungeon
    • END : indique la salle de sortie du dungeon
    • NONE : salle vide (sans portes)
    • ITEM=... : object a ramasser
  • Liste des objets :
    • KEY
    • ORANGE
    • BLUE
    • METAL
    • VIOLET
    • GREEN
    • RED
    • MAP
    • RADAR
    • SMALLTIME
    • BIGTIME
Les portes des salles ITEM sont déterminées a partir ce celles des salles DATA environnante. => Pas deux salles d'ITEM qui correspondent par une porte !
